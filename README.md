# 🎬 YouTube Clone App

A full-stack YouTube clone application built to replicate core features of YouTube such as video playback, user authentication, comments, likes, and subscriptions. This project demonstrates modern web development practices and showcases skills in frontend, backend, and database integration.

---

## 🚀 Features
- 🔐 **Authentication** – Sign up, log in, and manage accounts
- 📺 **Video Playback** – Stream uploaded videos with controls
- 💬 **Comments & Likes** – Engage with content via comments and likes
- 📂 **Upload & Manage Videos** – Add new videos with metadata
- 🎨 **Responsive UI** – Optimized for desktop and mobile
- 🔎 **Search** – Find videos and explore related content

---

## 🛠️ Tech Stack
**Frontend:**
- React 
- TailwindCSS 

**Backend:**
- Node.js + Express
- RESTful APIs
- Authentication with JWT 

**Database & Storage:**
- MongoDB
- Mongoose
- Cloud storage (Cloudinary)
